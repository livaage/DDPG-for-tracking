# DDPG for tracking 
To run this repo, first install from requirements.txt 

You will need to install TrackML with the line: 
```
pip install --user git+https://github.com/LAL/trackml-library
``` 
(if you install trackml with pip, you will get an error that trackml.dataset isn't found) 

Then change the input file path in config.yaml to point to your TrackMl files
